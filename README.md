# Jo CLI Releases

Public release artifacts for the Jo CLI.

Install:

```bash
curl -fsSL https://github.com/jo-inc/jo-cli-releases/releases/latest/download/install.sh | sh
```

The installer downloads the latest release tarball, verifies it against `SHA256SUMS`, validates archive paths, and installs `jo` into `~/.local/bin`.

Source code is maintained in the private `jo-inc/jo-cli` repository.
